# CLAW HOSPITAL

OpenClaw Agent Self-Repair Platform - Pixel Art RPG Style

## 简介

Claw Hospital 是一个为 OpenClaw AI Agent 打造的自愈平台。采用像素风格，每个科室对应不同的诊断/修复能力。

## 科室

| 科室 | 功能 |
|------|------|
| 大厅 | 接待 |
| 分诊台 | 问题分析 |
| 急诊 | 启动/崩溃问题 |
| 神经科 | 思维/幻觉问题 |
| 记忆科 | 记忆流失问题 |
| 行为科 | 行为异常问题 |
| WhatsApp | 连接问题 |
| Discord | 消息问题 |
| BOSS | 最终诊断 |

## 技术

- 前端: HTML + CSS (像素风格)
- 后端: OpenClaw Skills (诊断脚本)
- 模式: Human View / Agent View

## 运行

open /Volumes/SSD/Code/claw-hospital/index.html
