## 一、项目概述

OpenClaw 维修医院是一套面向甲壳类水族宠物（以龙虾为主）的智能健康管理与诊断系统。系统通过 AI 技术实现疾病预警、自动诊断、治疗方案推荐等功能，同时提供用户管理、病例记录、数据分析等完整业务支撑。

**设计原则:**
- **高可用性**: 系统需支持 7×24 小时运行，单点故障不影响核心业务
- **可扩展性**: 模块化设计，支持功能横向扩展与纵向深化
- **数据安全**: 医疗级数据保护，遵循隐私合规要求
- **AI 驱动**: 核心诊断能力由 AI 引擎提供，人工复核为辅

### 2.2 技术栈选型

| 层级 | 技术选型 | 选型理由 |
|------|----------|----------|
| **API 网关** | Kong / Nginx | 高性能、易扩展、插件生态丰富 |
| **服务框架** | Node.js (NestJS) / Go | 高并发、微服务友好 |
| **AI 推理** | Python + ONNX Runtime | 模型兼容性、推理性能 |
| **主数据库** | PostgreSQL | 事务支持、JSON 扩展、GIS 支持 |
| **缓存** | Redis Cluster | 高性能、会话存储、分布式锁 |
| **搜索引擎** | Elasticsearch | 全文检索、日志分析 |
| **对象存储** | MinIO (S3 兼容) | 自部署、成本控制 |
| **时序数据库** | TimescaleDB | 时序数据优化、SQL 兼容性 |
| **图数据库** | Neo4j | 知识图谱存储与查询 |
| **消息队列** | Apache Kafka | 高吞吐、持久化、流处理 |
| **容器化** | Docker + Kubernetes | 弹性伸缩、服务编排 |
| **监控** | Prometheus + Grafana | 可观测性、告警 |

### 4.2 核心 AI 模型设计

#### 4.2.1 视觉诊断模型

**任务:** 基于龙虾体表图像进行疾病检测与分类

**模型架构:**

```python
# 模型设计：针对甲壳类动物优化的 CNN 架构
# Backbone: EfficientNet-B4 (ImageNet 预训练，微调)
# Head: 多任务学习 (分类 + 定位 + 分割)

class LobsterDiagnosisModel(nn.Module):
    def __init__(self, num_diseases=15, num_keypoints=8):
        super().__init__()
        # 共享特征提取器
        self.backbone = EfficientNet.from_pretrained('efficientnet-b4')
        
        # 疾病分类头
        self.classifier = nn.Sequential(
            nn.AdaptiveAvgPool2d(1),
            nn.Flatten(),
            nn.Linear(1792, 512),
            nn.ReLU(),
            nn.Dropout(0.3),
            nn.Linear(512, num_diseases)  # 15 种常见疾病
        )
        
        # 病变区域定位头 (BBox)
        self.bbox_head = nn.Linear(1792, 4 * num_diseases)
        
        # 关键点检测头 (触角、鳃、尾等关键部位)
        self.keypoint_head = nn.Linear(1792, num_keypoints * 2)
        
        # 分割头 (病变区域像素级分割)
        self.decoder = UnetDecoder(1792, num_classes=1)
        
    def forward(self, x):
        features = self.backbone.extract_features(x)
        
        return {
            'disease_logits': self.classifier(features),
            'bbox': self.bbox_head(features),
            'keypoints': self.keypoint_head(features),
            'segmentation': self.decoder(features)
        }
```

#### 4.2.3 治疗推荐模型

**任务:** 基于诊断结果与病历历史，推荐最优治疗方案

**模型设计 - 强化学习方案:**

```python
# Deep Q-Network (DQN) for Treatment Recommendation
class TreatmentRecommender:
    """
    状态空间: 疾病类型 + 严重程度 + 并发症 + 虾的体型 + 过往用药史
    动作空间: 药品组合 + 剂量 + 用药周期 + 辅助措施
    奖励函数: 治愈率 - 成本 - 副作用评分
    """
    
    def __init__(self, state_dim, action_dim):
        self.q_network = nn.Sequential(
            nn.Linear(state_dim, 256),
            nn.ReLU(),
            nn.Linear(256, 256),
            nn.ReLU(),
            nn.Linear(256, action_dim)
        )
        
    def get_recommendation(self, state, top_k=3):
        q_values = self.q_network(state)
        # 返回 top-k 推荐，按 Q 值排序
        return torch.topk(q_values, top_k)
```
