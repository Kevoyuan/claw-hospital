# 小龙虾医院 - 视觉设计方案

## 1. 项目概述

**项目名称**：小龙虾医院 (Lobster Hospital)  
**项目类型**：宠物医疗机构品牌形象  
**目标用户**：小龙虾宠物主人、养殖业者  
**品牌调性**：专业、温馨、活泼、有趣

---

## 2. Logo 设计

### 核心元素

```
       🦞
      🏥
```

### 设计说明

- **主体图形**：简化的小龙虾轮廓，头部融合医院十字标志
- **字体风格**：圆润可爱的无衬线字体，"小龙虾"三字用活泼手写风，"医院"用稳重字体
- **配色**：主色调红色+白色，象征医疗与活力
- **应用场景**：门头、名片、社交媒体、App图标

### Logo 变体

| 场景 | 变体形式 |
|------|----------|
| 正式场合 | 完整Logo（图形+中英文全称） |
| 社交媒体 | 简化版（图形+中文简称） |
| 小尺寸应用 | 纯图形版（仅小龙虾+十字） |

## 3. 配色方案

### 主色调

| 颜色名称 | 色值 | 用途 |
|----------|------|------|
| **龙虾红** | `#E74C3C` | 主色，品牌识别度 |
| **珊瑚橙** | `#F39C12` | 强调色，活力点缀 |

### 辅助色

| 颜色名称 | 色值 | 用途 |
|----------|------|------|
| **医院白** | `#FFFFFF` | 背景、卡片 |
| **海盐蓝** | `#3498DB` | 链接、按钮 |
| **浅灰** | `#F8F9FA` | 分割线、禁用态 |
| **深灰** | `#2C3E50` | 正文文字 |

### 渐变方案

```css
/* 主题渐变 - 用于Banner和按钮 */
background: linear-gradient(135deg, #E74C3C 0%, #F39C12 100%);

/* 清新渐变 - 用于医疗区域 */
background: linear-gradient(180deg, #FFFFFF 0%, #F8F9FA 100%);
```

### 配色应用比例

- 主色（龙虾红）：40%
- 辅助色（蓝/橙）：20%  
- 白色背景：30%
- 文字/线条：10%

## 4. UI 风格

### 整体风格

**关键词**：萌趣、清新、专业、扁平化

### 设计原则

1. **圆润亲和** - 卡片、按钮、图标全部圆角设计，减少攻击性
2. **卡通趣味** - 适当使用小龙虾卡通形象，降低医疗的冰冷感
3. **清晰层级** - 重要信息用颜色/大小区分，层次分明
4. **留白呼吸** - 内容区域宽松，避免拥挤

### 组件规范

#### 按钮

```
主要按钮: 圆角12px, 背景#E74C3C, 白色文字
次要按钮: 圆角12px, 边框#E74C3C, 红色文字
禁用状态: 背景#BDC3C7, 不可点击
```

#### 卡片

```
圆角: 16px
阴影: 0 4px 12px rgba(0,0,0,0.08)
内边距: 20px
```

#### 图标风格

- 线性图标为主，线条粗细 2px
- 圆角端点，柔和友好
- 尺寸：24x24px（标准）、32x32px（突出）

#### 字体排版

```
标题: Noto Sans SC Bold, 24-32px
副标题: Noto Sans SC Medium, 18-20px
正文: Noto Sans SC Regular, 14-16px
小字: Noto Sans SC Regular, 12px
行高: 1.6
```

### 动画效果

- **页面切换**：淡入淡出 + 轻微上移（200ms ease-out）
- **按钮悬停**：轻微放大（scale 1.02）+ 阴影加深
- **卡片悬停**：上浮效果（translateY -4px）
- **加载动画**：小龙虾形状的脉动动画

## 7. 总结

小龙虾医院的视觉设计旨在打破传统医疗的冰冷印象，通过：

- **红色系主调** - 延续小龙虾本体色彩，增强品牌记忆
- **圆润柔和的UI** - 降低医疗严肃感，增加亲和力
- **小龙吉祥物** - 作为品牌人格化的核心，增强情感连接
- **统一的视觉语言** - 从Logo到图标到界面，保持高度一致

整体风格定位：**"专业但不高冷，医疗也可以很萌"**
