# Medical Records / 病历目录

This directory contains all collected issues and solutions organized by department.

## Structure

| File | Department | Description |
|------|------------|-------------|
| 01-emergency.md | Emergency | Startup/Crash issues |
| 02-neuro.md | Neuro | Thinking/Hallucination issues |
| 03-memory.md | Memory | Memory loss issues |
| 04-behavior.md | Behavior | Abnormal behavior issues |
| 05-discord.md | Discord | Discord message issues |
| 06-whatsapp.md | WhatsApp | Connection issues |
| 07-config.md | Config | Configuration issues |
| 08-model.md | Model | Model/API issues |
| 09-security.md | Security | Security vulnerabilities |
| 10-skill.md | Skill | Skill-related issues |

## Usage

Each file contains:
- Problem descriptions
- Error messages
- Solutions (commands to run)
- Severity levels
