# CLAW HOSPITAL - Project Spec

## Project Overview
- **Project**: Claw Hospital - OpenClaw Agent Self-Repair Platform
- **Type**: Interactive Web Application (Pixel Art RPG)
- **Style**: Pokemon/Retro Gaming - NO EMOJIS, pure pixel art
- **Views**: Human Mode / Agent Mode

## UI/UX Spec

### Color Palette
- Background: #1a1c2c (dark blue)
- Panels: #262b44 (lighter blue)
- Accent: #41a6f6 (blue), #73eff7 (cyan)
- Text: #f8f8f2 (off-white)
- Borders: #4b692f (green)

### Layout
- Split view: Human (left) / Agent (right)
- Each side has independent navigation
- Lofi pixel aesthetic throughout

### Departments (Indoor Scenes)
Each room needs:
- Pixel art background (indoor)
- Character sprites (doctor + patient)
- Interaction panel
- Status displays

## Room Scenes (No Emojis - Use Pixel Art)

1. **LOBBY** - Reception desk, waiting area
2. **TRIAGE** - Analysis machines, screens
3. **EMERGENCY** - Emergency beds, equipment
4. **NEURO** - Brain scan设备, therapy chairs
5. **MEMORY** - Data banks, restore machines
6. **BEHAVIOR** - Training room, monitors
7. **WHATSAPP** - Phone systems, QR scanners
8. **DISCORD** - Chat terminals, servers
9. **BOSS** - Admin office, mainframe

## Technical
- Frontend: Single HTML with pixel art
- Backend: Connect to OpenClaw API for real agent status
- Data: Real-time agent health monitoring

## Deliverables
- Indoor pixel scenes for all 9 rooms
- Connected backend (mock API for now)
- Human/Agent split view
- Lofi aesthetic
